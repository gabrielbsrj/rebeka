# Especificação do Projeto: teste_landpage

**Framework/Tech Stack**: nextjs
**Modelo Preferido**: claude-3-5-sonnet

## Descrição Funcional
Crie uma landing page moderna e profissional chamada "teste_landpage" com as seguintes características:

1. **Hero Section**: Título impactante, subtítulo descritivo, CTA (Call-to-Action) botão, imagem ou ilustração de destaque
2. **Features Section**: 3-4 cards destacando benefícios ou funcionalidades com ícones
3. **About/Sobre Section**: Breve descrição da empresa/produto/serviço
4. **Testimonials/Depoimentos**: 2-3 cards de depoimentos de clientes
5. **Pricing/Preços**: Seção de planos e preços (3 tiers)
6. **FAQ Section**: Perguntas frequentes com acordeão (accordion)
7. **Contact/Contato**: Formulário de contato ou informações
8. **Footer**: Links úteis, redes sociais, copyright

**Design Requirements**:
- Design responsivo (mobile-first)
- Paleta de cores moderna (sugira azul escuro + tons de cinza ou escolha uma paleta profissional)
- Animações suaves com Framer Motion
- Tipografia clean e legível
- Componentes reutilizáveis
- SEO otimizado
- Performance otimizada

**Tecnologias**:
- Next.js 14+ com App Router
- TypeScript
- Tailwind CSS
- Framer Motion para animações
- Lucide React para ícones
- shadcn/ui para componentes base

Entregue o projeto completo, funcional e pronto para deploy na Vercel.
