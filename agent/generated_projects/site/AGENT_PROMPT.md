# Especificação do Projeto: site

**Framework/Tech Stack**: nextjs
**Modelo Preferido**: claude

## Descrição Funcional
Crie um site jurídico completo e profissional para o Gabriel. O site deve incluir: 1) Homepage com apresentação do escritório de advocacia, 2) Página de serviços jurídicos (direito civil, criminal, trabalhista, família, empresarial), 3) Página de sobre nós com história do escritório, 4) Página de blog jurídico com artigos, 5) Formulário de contato, 6) Sistema de agendamento de consultas, 7) Área de clientes com portal de acesso, 8) Integração com WhatsApp Business, 9) SEO otimizado para termos jurídicos, 10) Design profissional em tons de azul e dourado, 11) Responsivo para mobile, 12) SEO com sitemap.xml e robots.txt, 13) Integração com Google Analytics, 14) Sistema de comentários e reviews, 15) Página de política de privacidade e termos de uso
